Термины, которые я выучил

Тег del
Это тег отвечающий за зачеркивание текста
Так же у этого тега есть аналог в виде тега s
Тег HTML
Этот тег прописывается в самом начале документа для верного синтаксиса. Без него работает документ но по правилам верстки будет верно его прописывать
Тег em
Этот тег служит для преобразоования текста в вид курсива

У этого тега так же есть аналог в виде тега i

Тег table
Этот тег используется для работы с таблицами и является парным тегом
Тег meta
Тег мета используется для внесения различыных метаданных для корректной работы сайта
Размещается внутри тега head
